# Java CRUD Application with MySQL (JDBC)

This project is a simple **Java application** that performs **CRUD operations** (Create, Read, Update, Delete) using **MySQL** database through **JDBC connectivity**.  
It is designed for beginners to understand Core Java, JDBC, and how to connect Java with a database.
